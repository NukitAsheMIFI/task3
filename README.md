Task 3: Сортировка строк файла с поддержкой кириллицы

Описание

Данный проект реализует чтение строк из файла, их сортировку различными способами и запись результата в выходной файл на языке C.

Программа поддерживает два типа сортировки: лексикографическую (с учётом регистра и кириллицы) и "простую" (побайтовое сравнение). Оба типа могут выполняться в прямом или обратном порядке.

Функциональные возможности

Чтение из файла: программа считывает весь текст из входного файла и разбивает его на строки по символу переноса строки.

Фильтрация пустых строк: строки, состоящие только из пробельных символов, игнорируются при сортировке.

Динамическое выделение памяти: массив строк динамически расширяется при необходимости с использованием realloc.

Четыре режима сортировки:
- plain — прямая сортировка (побайтовое сравнение)
- rplain — обратная сортировка (побайтовое сравнение)
- lex — прямая лексикографическая сортировка
- rlex — обратная лексикографическая сортировка

Поддержка кириллицы: лексикографическая сортировка корректно обрабатывает символы кириллицы с использованием tolower.

Запись в файл: отсортированные строки записываются в выходной файл, каждая строка с новой строки.

Стек технологий

Язык программирования: C (стандарт C11).

Библиотеки: stdio.h, stdlib.h, string.h, ctype.h.

Система сборки: GNU Make.

Компилятор: GCC.

Инструкция по сборке и запуску

Сборка проекта

Для компиляции исходного кода в исполняемый файл необходимо выполнить команду:

make

В результате сборки будет создан файл task3.

Запуск программы

Программа принимает три обязательных аргумента:

./task3 входной_файл выходной_файл режим_сортировки

Примеры запуска

Прямая сортировка (побайтовое сравнение):

./task3 input.txt output.txt plain

Обратная сортировка (побайтовое сравнение):

./task3 input.txt output.txt rplain

Прямая лексикографическая сортировка:

./task3 input.txt output.txt lex

Обратная лексикографическая сортировка:

./task3 input.txt output.txt rlex

Формат входного файла

Входной файл должен содержать строки текста, разделённые символами переноса строки. Пустые строки или строки, состоящие только из пробелов, игнорируются.

Пример входного файла (input.txt):

яблоко
банан
апельсин
груша

Пример выполнения:

./task3 input.txt output.txt lex

Содержимое выходного файла (output.txt):

апельсин
банан
груша
яблоко

Очистка временных файлов

Для удаления объектных файлов и исполняемого файла выполните:

make clean

Технические подробности реализации

Чтение файла: программа определяет размер файла с помощью fseek и ftell, выделяет буфер необходимого размера и считывает содержимое целиком.

Разбиение на строки: используется strtok с разделителем "\n" для получения отдельных строк.

Проверка пустых строк: функция is_empty проверяет, состоит ли строка только из пробельных символов (пробел, табуляция, возврат каретки и т.д.).

Динамический массив: изначально выделяется память на 5 указателей. При заполнении массива размер удваивается с помощью realloc.

Алгоритм сортировки: используется сортировка пузырьком. Направление сортировки регулируется параметром dir (1 — прямой порядок, -1 — обратный).

Лексикографическая сортировка: функция lex сравнивает строки посимвольно с приведением к нижнему регистру через tolower. Для корректной работы с кириллицей используется приведение к unsigned char.

Простая сортировка: функция plain сравнивает строки побайтово без учёта регистра.

Освобождение ресурсов: в конце работы программы освобождаются выделенная память и закрываются файловые дескрипторы.
